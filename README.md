# Stereo-seq_in_ALS_human_spinal_cord
Master's thesis project in collaboration with Karolinska Institutet. 
Stereo-seq is an high-resolution spatial transcriptomics method developed by BGI. 

The codes are focused on transcriptomic profiling and niche analysis of manually segmented lower motor neurons (LMNs) from stereo-seq slides.

Repository will be populated right after the graduation date (end of february 2026)
